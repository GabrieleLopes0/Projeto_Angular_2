Sistema de Gerenciamento de Consultores

📋 Descrição

Este é um sistema completo desenvolvido em Angular para gerenciar consultores de uma empresa. A aplicação implementa autenticação, controle de acesso baseado em perfis, e operações completas de CRUD (Create, Read, Update, Delete).

🏗️ Arquitetura

📦 Services: Gerenciamento de estado e lógica de negócio (AuthService, ConsultantService)
🛡️ Guards: Proteção de rotas com AuthGuard
📥 @Input(): Comunicação Pai → Filho para enviar dados
📤 @Output(): Comunicação Filho → Pai usando EventEmitter
💉 Dependency Injection: Injeção de dependências do Angular
📝 Reactive Forms: Formulários com validação robusta
